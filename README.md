## Usage

The project is configured for `run`, `test` and `build` each day independently.

For example:

    # only run the day01
    $ cargo run --bin day01

    # only test the day02
    $ cargo test --bin day02

    # only build the day03
    $ cargo build --bin day03

Folder structure:

     src
     └── day01
         ├── bin
         │   ├── day01.rs
         │   ├── part1.rs
         │   └── part2.rs
         ├── input.txt
         └── README.md
